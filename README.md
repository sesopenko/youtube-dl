# sesopenko/youtube-dl

A docker project for downloading videos from youtube.

## Requirements

* Windows (tested on 11, will probably work on 7 and 10)
* Docker for desktop (windows)

## Usage

build:

```
build
```

get formats:

```
formats <youtube-url>
```

download:

```
download <format> <youtube-url>
```

files are stored in out/
